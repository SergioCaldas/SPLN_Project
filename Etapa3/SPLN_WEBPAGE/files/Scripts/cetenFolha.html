CetenFolha.pl

Esta script faz um reconhecimento de todas as palavras e caracteres que aparecem nas noticias.

Posteriormente calcula as suas respectivas frequencias.

Calculo das frequencias

Neste excerto de codigo sao calculadas as frequencias.

    foreach my $name (sort keys %paises){
      if($paises{$name} > 0){
        my $freq = ($paises{$name}*1000000)/$total_palavras;
        $freq = sprintf("%.4f",$freq);
        my $log = log10($paises{$name});
        $log = sprintf("%.4f",$log);
        print "$name,$paises{$name},$freq,$log\n";
      }
    }

Funcao logaritmo

Funcao que calcula o logaritmo base 10 de n

      sub log10{
        my $n = shift;
        return log($n)/log(10);
      }
