Paises.pl

Esta script faz um reconhecimento dos paises que aparecem nas noticias.

Alem disso, verifica se esses paises existem na Hash respectiva onde se encontram todos os paises presentes na Base de Dados.

Posteriormente calcula as suas respectivas frequencias.

Calculo das frequencias

Neste excerto de codigo sao calculadas as frequencias dos paises.

      foreach my $pais (sort keys %paises){
        if($paises{$pais} > 0){
          my $freq = ($paises{$pais}*1000000)/$total_palavras;
          $freq = sprintf("%.4f",$freq);
          my $log = log10($paises{$pais});
          $log = sprintf("%.4f",$log);
          print "$pais,$paises{$pais},$freq,$log\n";
        }
      }
